
# oServ - Gestão Ordem de Serviço

## Descrição

O **oServ - Frontend** é a interface de usuário desenvolvida para a aplicação de gestão de ordens de serviço. Utiliza o framework **Angular** para criar uma interface web interativa e dinâmica, integrando com o backend para realizar operações como cadastro, visualização e gerenciamento das ordens de serviço.

## Tecnologias Utilizadas

* **Angular** 17.x: Framework para desenvolvimento de aplicações web
* **Bootstrap** 5.x: Framework de UI para estilos responsivos e componentes
* **Ngx-Bootstrap** : Componentes Bootstrap com suporte para Angular
* **RxJS** : Biblioteca para programação reativa
* **CanvasJS** : Biblioteca de gráficos para visualizações interativas
* **Karma** e  **Jasmine** : Frameworks para testes unitários
