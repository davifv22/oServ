# oServ - Gestão Ordem de Serviço

## Descrição

O **oServ - Backend** é uma aplicação backend desenvolvida para gerenciar ordens de serviço. O sistema permite o gerenciamento e controle de processos relacionados à prestação de serviços, integrando-se com banco de dados MySQL utilizando o Sequelize como ORM.

## Tecnologias Utilizadas

* **Node.js** com **TypeScript**
* **Express** para criação da API RESTful
* **Sequelize** para comunicação com banco de dados MySQL
* **Winston** para logging de eventos
* **MySQL2** para comunicação com o banco de dados
* **TypeScript** para maior escalabilidade e manutenção
* **Cors** para gerenciar requisições de diferentes origens
